About This Blog

A Django-powered blog blending manual and AI-generated content.

Project Overview

This is a full-stack web application built with Django, showcasing my skills in Python, web development, and AI integration. It’s a personal blog creation platform where users can create posts manually or generate them using Google’s Gemini API. I designed it to learn Django and to experiment with AI APIs.

Key Features

  • User authentication (login, register, logout)
  • Manual blog post creation, editing, and deletion
  • AI-generated blog posts with SEO optimization
  • Dynamic sidebar with latest posts
  • Pagination and responsive design with Bootstrap

Tech Stack

  • Backend: Django (Python), SQLite
  • Frontend: Bootstrap 4, Custom CSS
  • AI: Google Gemini API
  • Tools: Git, VS Code
